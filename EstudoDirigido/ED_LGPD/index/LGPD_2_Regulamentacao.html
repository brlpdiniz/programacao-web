Lei Geral de Proteção de Dados Pessoais (LGPD)


A Lei Geral de Proteção de Dados (LGPD) é uma lei brasileira que entrou em vigor em setembro de 2020. Tendo como objetivo proteger os direitos fundamentais de privacidade e liberdade dos cidadãos, estabelecendo regras para o tratamento de dados pessoais pelas empresas e organizações.



Direitos


A LGPD garante aos titulares de dados uma série de direitos em relação ao tratamento de seus dados pessoais. Esses direitos incluem:

  • Direito de acesso: o titular dos dados tem o direito de saber quais dados pessoais estão sendo coletados, para que finalidade e como são tratados.
  • Direito de correção: o titular dos dados tem o direito de corrigir dados pessoais incorretos ou incompletos.
  • Direito de exclusão: o titular dos dados tem o direito de solicitar a exclusão de seus dados pessoais, salvo nas hipóteses em que a LGPD permite o tratamento dos dados sem o consentimento do titular.

  • Direito de oposição: o titular dos dados tem o direito de se opor ao tratamento de seus dados pessoais para determinadas finalidades, como marketing direto.

Responsabilidades


A LGPD define que as empresas e organizações que tratam dados pessoais devem cumprir uma série de obrigações e responsabilidades. Algumas das principais são:

  • Nomeação de um Encarregado de Proteção de Dados (DPO): Essa é uma função importante criada pela LGPD, que consiste em indicar uma pessoa ou equipe responsável por garantir que a organização esteja em conformidade com a lei e que os direitos dos titulares de dados sejam respeitados.
  • Direito de oposição: O titular dos dados tem o direito de se opor ao tratamento de seus dados pessoais para determinadas finalidades, como marketing direto.
  • Registro das atividades de tratamento: Todas as atividades de tratamento de dados pessoais devem ser registradas e documentadas, de forma a permitir a verificação e a auditoria do cumprimento da LGPD.
  • Comunicação de incidentes: Em caso de incidentes de segurança que possam comprometer a privacidade ou a segurança dos dados pessoais, as empresas e organizações devem notificar as autoridades competentes e os titulares dos dados afetados.
