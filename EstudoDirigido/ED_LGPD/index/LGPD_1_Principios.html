Lei Geral de Proteção de Dados Pessoais (LGPD)


A Lei Geral de Proteção de Dados (LGPD) é uma lei brasileira que entrou em vigor em setembro de 2020. Tendo como objetivo proteger os direitos fundamentais de privacidade e liberdade dos cidadãos, estabelecendo regras para o tratamento de dados pessoais pelas empresas e organizações.



Princípios


A Lei Geral de Proteção de Dados (LGPD) estabelece um conjunto de princípios que norteiam o tratamento dos dados pessoais no Brasil. Esses princípios foram criados com o intuito de garantir a proteção dos direitos fundamentais de privacidade e liberdade dos cidadãos. A lei exige que todas as empresas e organizações que coletam, armazenam e processam dados pessoais cumpram com esses princípios, o que é fundamental para garantir a transparência e a segurança no uso das informações dos indivíduos. Nesta tabela, vamos conhecer em detalhes os princípios da LGPD e entender a sua importância para a proteção destes dados.



Princípio Descrição
Finalidade Os dados pessoais devem ser coletados para propósitos específicos, explícitos e legítimos, e não podem ser usados para outros fins sem o consentimento do titular dos dados
Adequação A coleta de dados pessoais deve ser adequada, relevante e limitada ao mínimo necessário para atender à finalidade para a qual foram coletados
Necessidade Os dados pessoais devem ser coletados somente na medida em que forem necessários para a finalidade pretendida
Livre acesso Os titulares dos dados têm direito a informações claras, precisas e acessíveis sobre o tratamento de seus dados pessoais
Qual. Dados Os dados pessoais devem ser exatos, atualizados e, quando necessário, corrigidos
Transparência As empresas e organizações devem informar aos titulares de dados sobre o tratamento de seus dados pessoais
Segurança Medidas técnicas e administrativas devem ser adotadas para garantir a segurança dos dados pessoais e protegê-los contra acessos não autorizados, perda, destruição ou danificação
